## Game Description
Welcome to Floppy Disc a game where you play as a flying floppy disc that lost its bits. Collect "Bit Coins" to gain access to the next section of the system, but wait there has been a security breach and a virus is now chasing you! Press space to flop away to safety, bounce against walls, gain those bits and navigate the increasingly complex system architecure.

| Action | Description |
| --- | --- |
| Pressing Spacebar | The floppy disc will fly upwards |
| Hitting wall | Bounce and change direction|
| Touch bit coins | Collect bit coins and open the gate if all are collected|
| Touch Virus | Floppy disc will flop and die |
